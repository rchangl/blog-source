# 音高记谱法

## 科学音高记谱法 Scientific pitch notation

中央 C 为 C4

钢琴最左边为 A0  
钢琴最右边为 C8

## 赫姆霍兹音高记法 Helmholtz Pitch Notation

中央C = c'

C5 = c''
C4 = c'  
C3 = c  
C2 = C  
C1 = C, (or ,C or C⸜)

## 唱谱音高

非国际通用，中国

中央C = 女c1 = 男c2

C5 = 女c2 = 男c3  
C4 = 女c1 = 男c2  
C3 = 女c = 男c1  
C2 = 女C = 男c  
C1 = 女C1 = 男C

## MIDI 音高

类似科学音高，但每个厂商可能有所不同
